Sample Integration Code

Request Example (JavaScript)
const apiKey = 'YOUR_API_KEY';
const requestBody = {
  customer_id: 'A123456789',  // Can also be a hashed value
  marketplace_id: 'AMZN_RETAIL_US',
  request_id: uuidv4(),
  order_value: 249.99,
  item_quantity: 3,
  basket_size: 127.50,
  high_value_flag: true,
  device_id: 'd8e8fca2dc0f896fd7cb4cb0031ba249'
};

// For a numeric score (0-100)
fetch('https://api.amazonbap.com/v1/score-vending/numeric-score', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${apiKey}`
  },
  body: JSON.stringify(requestBody)
})
.then(response => response.json())
.then(data => console.log(data));

// For a binary decision (allow/deny)
fetch('https://api.amazonbap.com/v1/score-vending/binary-decision', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${apiKey}`
  },
  body: JSON.stringify(requestBody)
})
.then(response => response.json())
.then(data => console.log(data));

Security & Privacy

Data Privacy Protections

  • All API requests are encrypted in transit using TLS 1.3
  • Customer IDs can be hashed before transmission for additional privacy
  • Stateless processing mode available - no data retention
  • Compliance with GDPR, CCPA, and HIPAA requirements
  • All API calls are logged for audit purposes with configurable retention periods
